# 🚀 PromptForge AI

**PromptForge AI** is an advanced AI-powered platform for mastering prompt engineering through interactive challenges, real-time evaluation, and personalized learning paths.

---

## ✨ Features

### 🎯 Core Modes
- **Problem Mode**: Solve AI/System Design problems with prompt engineering
- **Idea to Prompt**: Transform creative ideas into optimized prompts
- **Technical Challenge**: DSA, DBMS, and Algorithm design challenges

### 📊 Advanced Analytics
- Real-time prompt evaluation with AI scoring
- Detailed performance metrics and radar charts
- Constraint tracking (word/token limits)
- Historical progress tracking

### 🎨 User Experience
- Dark/Light theme toggle
- Responsive design for all devices
- Interactive dashboards
- PDF report generation
- Badge system and achievements

---

## 🛠️ Tech Stack

### **Frontend**
- React 18 + Vite
- TailwindCSS for styling
- Framer Motion for animations
- Recharts for data visualization
- Axios for API communication

### **Backend**
- FastAPI (Python)
- MongoDB for database
- OpenAI GPT-3.5/4 for AI evaluation
- JWT authentication
- Pydantic for data validation

---

## 📦 Installation

### **Prerequisites**
- Node.js 18+ and npm
- Python 3.9+
- MongoDB
- OpenAI API Key

---

### **Backend Setup**

```bash
# Navigate to backend
cd backend

# Create virtual environment
python -m venv venv

# Activate virtual environment
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Create .env file
cp .env.example .env

# Edit .env with your credentials:
# MONGODB_URL=mongodb://localhost:27017
# DATABASE_NAME=promptforge
# JWT_SECRET_KEY=your-secret-key-here
# OPENAI_API_KEY=your-openai-api-key
# FRONTEND_URL=http://localhost:5173

# Run the server
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
